### 使用
通过[launch-cli](https://github.com/Iixianjie/launch-cli)安装


- 列出了所有开发中可能会用到的配置(就个人而言)
- 更方便的配置环境变量且不用必须添加VUE_APP_前戳或新建.env文件
- 配置别名选项 如: ~sass  ~component ~src
- 配置是否在prod阶段清除console
- 快速配置需要引入的sass全局变量、mixin等
- gizp
- 自动化的基础组件和模块导入
- 清晰的文件结构
- 基于koa的简单mock服务
